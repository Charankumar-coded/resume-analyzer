# ResumeIQ - AI Resume Analyzer

A polished single-page resume review tool with a modern dark-themed UI/UX, built to analyze resumes against job descriptions, calculate ATS matching scores, and break down relevant keywords.

---

## 🚀 Features

- **Responsive Dark Dashboard:** Clean, professional interface with a dark-themed layout.
- **Real-Time Word Tracking:** Input editors for both your resume and job description with automatic updates.
- **ATS Match Score:** Instantly calculates an analysis percentage based on target requirements and matching metrics.
- **Keyword Analyzer:** Visually breaks down matching, partial, or missing technical skills.
- **Local Storage Drafts:** Automatically saves your work locally in the browser so you never lose your text on accidental refreshes.
- **Export Reports:** Quickly copy summaries or download raw JSON analytical data.

---

## 🛠️ How to Use

1. Open `index.html` in any web browser.
2. Paste your **Resume** text into the left editor panel.
3. Paste the **Job Description** or target keywords into the right editor panel.
4. Click **Analyze Match** to run the local matching engine.
5. *(Optional)* Provide your AI API Key in the settings field to unlock deeper LLM insights.
